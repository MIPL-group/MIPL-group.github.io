Course Profile

Introduction to Machine Learning

Theory and implementation of state of the art machine learning algorithms for large-scale real-world applications. Topics include supervised learning (regression, classification, kernal methods, neural networks, and reg... ularization) and unsupervised learning, (clustering, density estimation, and dimensionality and reduction).  Show more >
